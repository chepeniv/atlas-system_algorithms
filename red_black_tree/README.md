## Requirements

- compilation: `gcc -Wall -Werror -Wextra -pedantic`
- a "readme" at the root of the project folder is mandatory
- code and documentation should be betty-compliant
- global variables are _not_ allowed
- no mare than five functions per file
- the standard library is _allowed_
- include all function prototypes should be included in the `rb_trees.h` header
  file
- all header files should be include guarded
